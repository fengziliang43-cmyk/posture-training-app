# Project Rules

## Project Goal

This project designs and later implements a personal posture-and-training app for Liang. The app should prioritize safety, posture, fat loss, muscle gain, then conditioning.

## Current Phase

The current phase is implementation planning. Do not implement app code until the implementation plan is reviewed and approved.

## Directory Rules

- `docs/superpowers/specs/`: approved design specs.
- `docs/superpowers/plans/`: implementation plans.
- `.superpowers/`: temporary visual brainstorming mockups and state.
- App code should not be added until the spec and implementation plan are approved.

## Privacy Rules

- Do not copy body photos into the repository unless Liang explicitly asks.
- Do not commit private photos, health data, secrets, API keys, or credentials.
- Treat posture and health notes as private local project data.

## Training Safety Rules

- This app is not medical diagnosis or treatment.
- Liang has reported S-shaped scoliosis, with the lumbar curve around 30 degrees from an older X-ray.
- Training recommendations should prioritize conservative progression, pain feedback, core stability, hip control, and avoiding aggressive loading when pain or fatigue is high.
